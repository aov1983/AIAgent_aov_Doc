# AI Agent for Requirements Analysis

## Описание
Специализированный агент для автоматизации анализа неформализованных документов, декомпозиции их на атомарные единицы знаний и сохранения в базу знаний (RAG/Graph).

## Возможности
- Парсинг документов (MS Word, текст, таблицы, изображения)
- Атомарная декомпозиция на смысловые блоки
- Извлечение метаданных (риски, критичность, тип исполнителя)
- Поиск похожих требований в RAG
- Выявление дубликатов и противоречий
- Поддержка 3-4 AI моделей через API

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```python
from agent import RequirementsAgent

# Инициализация агента
agent = RequirementsAgent(api_key="your-api-key")

# Загрузка и анализ документа
result = agent.process_document("requirements.docx")

# Получение отчета
print(result.report)
```
